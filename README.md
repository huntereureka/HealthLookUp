## Installation
Dowload the files and unzip them, afterwards open the folder up in your favorite IDE!  
Libraries you will **need** to install:  
`pip3 install flask`  
`pip3 install requests`  
`pip3 install beautifulsoup4`  
`pip3 install fuzzywuzzy`
